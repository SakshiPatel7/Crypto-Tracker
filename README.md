# Real-Time Crypto Tracker

Tracks live prices of 5 cryptocurrencies using mocked updates and Redux Toolkit.

## 📦 Tech Stack

- React + Vite
- Redux Toolkit
- Tailwind CSS

## 🔧 Setup

```bash
git clone <your-repo>
cd crypto-tracker
npm install
npm run dev
```
